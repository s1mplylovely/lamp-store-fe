## Запуск локально

1. Клонировать репозиторий

```bash
git clone https://github.com/s1mplylovely/lamp-store-fe.git
cd lamp-store-fe
```

2. Установить зависимости

```bash
npm install
```

3. Запустить dev-сервер

```bash
npm run dev
```

Приложение откроется по адресу [http://localhost:5173](http://localhost:5173).
